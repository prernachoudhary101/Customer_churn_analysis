# Customer_churn_analysis

## Overview
This project analyzes telecom customer behavior to identify drivers of customer churn and build predictive models.

## Tools Used
Python
Pandas
Scikit-learn
SQL
Power BI

## Dataset
Telco Customer Churn Dataset with 7,000+ customer records.

## Key Insights
- Customers with month-to-month contracts churn the most
- Customers with higher monthly charges show higher churn probability
- Low tenure customers are more likely to churn

## Model
Logistic Regression model with ~82% accuracy.

## Dashboard
Power BI dashboard showing churn trends and customer segments.
